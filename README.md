# SSRImporter
SSR URL batch import tool

## macOS ssr链接批量导入工具

* 预览
![preview](images/preview_importer.png)
* 因为mac的小飞机没有重粘贴板批量导入功能，所以简单做了一个小工具
* 原理
	* 循环遍历输入字符串，利用终端打开浏览器，导入节点
* 如有更好方案，欢迎push
* 用法:每个ssr链接以","隔开
